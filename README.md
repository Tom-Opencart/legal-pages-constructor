# ⚖️ Конструктор юридических документов для OpenCart

**Конструктор документов по 152-ФЗ, 38-ФЗ и ГК РФ** — бесплатный браузерный инструмент для быстрой генерации юридически корректных текстов страниц интернет-магазина.

🔗 **[Открыть конструктор онлайн →](https://tom-opencart.github.io/legal-pages-constructor/)**

---

## 📋 Что генерирует конструктор

| Документ | Основание |
|---|---|
| Политика обработки персональных данных | 152-ФЗ |
| Публичная оферта | ГК РФ ст. 435–443 |
| Согласие на обработку персональных данных | 152-ФЗ |
| Согласие на получение рассылки | 38-ФЗ «О рекламе» |
| Политика использования Cookie | 152-ФЗ / ePrivacy |
| Пользовательское соглашение | ГК РФ |

---

## 🚀 Возможности

- **Мгновенная генерация** — заполните реквизиты один раз, шаблоны обновляются в реальном времени
- **Минималистичный HTML-вывод** — без лишних стилей, готов к вставке в любой OpenCart-шаблон
- **Копирование одной кнопкой** — иконка копирования зафиксирована при прокрутке
- **Автосохранение** — данные сохраняются в localStorage, не теряются при перезагрузке
- **Встроенная справка** — подробная инструкция по каждому документу и интеграции с OpenCart
- **Только браузер** — никакого сервера, никакой регистрации, работает офлайн

---

## 🏪 Интеграция с OpenCart

Конструктор создаёт чистый HTML, предназначенный для вставки в страницы **Информация** (Каталог → Информация):

1. Открой нужный документ в конструкторе
2. Нажми иконку копирования в правом верхнем углу превью
3. Перейди в Админ → Каталог → Информация → открой нужную страницу
4. Вставь скопированный HTML в поле редактора (режим «Источник»)
5. Сохрани
